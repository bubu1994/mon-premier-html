HTML (HyperText Markup Language)
HTML est un language de balisage utilisé pour structurer le contenu des pages web. Une balise est un mot-clef entouré de chevrons. La plupart fonctionnent par paires (une balise ouvrante et une balise fermante).
Exemple d'un fichier html 

	<!DOCTYPE html>
	<html>
		<head>
			<meta charset="utf-8">
			<title>Bonjour</title>
		</head>
	<body>
		<p>Bonjour</p>
	</body>
	</html>
  • <!DOCTYPE html> : indique qu’on utilise HTML5.
  • <html> : racine du document HTML.
  • <head> : section contenant les métadonnées (titre, encodage, etc.).
  • <meta charset="UTF-8"> : définit l’encodage. UTF-8 englobe tous les caractères.
  • <title> : le texte affiché dans l’onglet du navigateur.
  • <body> : le contenu visible de la page.
  • <p> : 'p' = paragraphe. Indique au navigateur que le contenu qu'elle englobe doit être affiché comme un bloc de texte distinct.

Pour exécuter (ou plutôt afficher) un fichier HTML, il suffit de l’ouvrir dans un navigateur web.
  1. Double-cliquer sur le fichier avec l'extension html.
  2. Clic droit > Ouvrir avec un navigateur web (firefox, chrome, etc...)
  3. Depuis un éditeur comme VS Code, installer l'extension Live Server. Clic-droit sur le fichier et open with Live Server. La page se rechargera automatiquement à chaque modification.

On peut voir le code d'un fichier html en appuyant sur F12 ou clique-droit et Inspecter.

Quelques balises
<br> = break. Insère un retour à la ligne.
<hr> = horizontal rule. Crée une ligne horizontale.
<strong> met le texte en gras. Indique que ce texte a une importance particulière.
<del> = delete. Indique au navigateur que le texte n'est plus pertinent et à supprimer.
<mark> marque le texte
<pre> = preformated text. 
	Affiche le texte exactement tel qu'il est écrit dans le code source,
	en conservant les espaces, tabulations et sauts à la ligne.
<abbr> = abbreviation. HTML
<summary><details> cache un texte
<i> = italique
<q> = quote
<cite> met le texte en italique. Sert à citer un document, film, etc

La sémantique et la structure du code sont essentielles en HTML.

Définition de sémantique: Étude du sens, de la signification des signes, notamment dans le langage.

Les moteurs de recherche comprennent mieux la hiérarchie et l'importance du contenu grâce aux balises sémantiques ce qui améliore le référencement dans les résultats de recherche. Par exemple, <i> et <cite> ont le même rendu visuel, mais ne signifient pas la même chose, ne seront pas interprétés de la même manière par les navigateurs.

Un code bien structuré avec des balises appropriées facilite la compatibilité entre différents navigateurs, appareils et tailles d'écran, ainsi que la maintenabilité du code, c'est-à-dire la lecture, la compréhension et la modification du code par les développeurs, réduisant les erreurs et accélérant le développement.

CSS (Cascading Style Sheets)
CSS est un langage qui sert à définir le swag style et la mise en forme des pages HTML : couleurs, polices, marges, disposition, etc.
Intégrer un fichier CSS au fichier HTML
À l'intérieur de la balise <head>, utiliser la balise <link rel="stylesheet" href="mon_fichier.css"> 

	<head>
		<meta charset="utf-8">
		<title>Mon premier site web</title>
		<link rel="stylesheet" href="mon_fichier_style.css">
	</head>
  • <link> permet d'associer un fichier externe de style au document HTML
  • L’attribut rel="stylesheet" (rel = relation) précise qu’il s’agit d’une feuille de style.
  • L’attribut href="mon_fichier.css" (hypertext reference) indique le chemin vers le fichier CSS.
Exemples
Le CSS utilise un système de sélecteurs et de propriétés. Les sélecteurs identifient quels éléments HTML styliser. Les propriétés définissent comment ces éléments doivent apparaître.
Exemple 1: Sélecteur d'élément.
Les propriétés 'background-color' et 'margin' définies dans p{} s'appliquent à toutes les balises <p> 

	p {
		/*propriété: valeur*/
		background-color: blue;
		margin: 30px;
	}
Exemple 2: Sélecteur de classe.
Il cible tous les éléments HTML ayant class="gras-et-rouge"
Dans fichier CSS:

	.gras-et-rouge {
		font-weight: bold;
		color: red;
	}
Sélecteur de classe avec l'élément html 'span' dans fichier HTML: 

	<span class="gras-et-rouge">
		Sélecteur de classe avec l'élément html 'span' dans fichier HTML:
	</span>
Exemple 3: Sélecteur d'identifiant (ID).
L'utilisation doit être unique, contrairement au sélecteur de classe.
Utilisez un ID si vous voulez proposer une navigation vers des sections de votre site. 

	#style-unique {
		font-family: 'Fredericka the Great', cursive;
		font-size: 1.5em;
		color: rgb(163, 124, 25);
	}
Sélecteur d'ID avec l'élément html 'span' dans fichier HTML: 

	<span id="style-unique">
		Sélecteur d'ID avec l'élément html 'span' dans fichier HTML:
	</span>
Exemple 4: Sélecteurs combinés.
Sélecteurs qui associent plusieurs éléments en précisant leur relation dans le document HTML.

  1. Sélecteur combiné de classe. 
    
	div.title {
		background-color: bisque;
		padding: 15px;
		border-radius: 10px;
		border: 5px double rgb(51, 51, 17);
		font-weight: bold;
	}

	div.title.second {
		background-color: rgb(148, 148, 239);
	}
    Le sélecteur div.title.second cible tous les éléments <div> ayant à la fois les classes title et second. 
    
	<div class="title second">Titre secondaire</div>
    La classe title.second "hérite" de la classe title. Ayant une plus grande spécifité, sa propriété background-color écrase l'autre.
  2. Sélecteur descendant. 
    
	div p {
		color: blue;
	}
    Tous les <p> à l'intérieur d’un <div> seront bleus, peu importe le niveau de profondeur.
  3. Sélecteur enfant direct. 
    
	ul > li {
		list-style: square;
	}
    Cible uniquement les <li> qui sont enfants immédiats de <ul>.
  4. Sélecteur de frère adjacent. 
    
	h1 + p {
		margin-top: 0;
	}
    Cible le premier <p> qui suit un <h1>.
  5. Sélecteur général de frères. 
    
	h1 ~ p {
		color: gray;
	}
    Tous les <p> qui suivent un <h1> au même niveau seront gris.
